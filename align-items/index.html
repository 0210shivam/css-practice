Align Items Property

The align-items property in CSS is used in flexbox layouts to control how flex items are aligned along the cross axis of the flex container. It applies to the children (flex items) of a flex container and accepts various values to specify how the items should be aligned vertically.

Some properties are - stretch: this will not work with flex-wrap, flex-start, flex-end, center, baseline.


Example of Align Items Applied -

align-item: center and align-content: center
