Align Content Property

It's used in flexbox layouts to align the flex container's lines within the flex container when there's extra space in the cross-axis. This property has effect only when the flex container has multiple lines of flex items.

Some properties are - flex-start, flex-end, center, space-between, space-around, stretch.


Example of Align Content Applied -

align-content: center
